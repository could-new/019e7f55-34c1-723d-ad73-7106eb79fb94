# ShopSphere

A modern e-commerce platform built with Flutter.

## Features

- **Product Browsing**: Discover featured and trending products.
- **Search & Filter**: Easily find products using search and category filters.
- **Responsive Design**: Works seamlessly on mobile and web platforms.
- **Modern UI**: Clean and premium user interface with smooth animations.

## Tech Stack

- **Frontend**: Flutter (Android, iOS, Web)
- **Design**: Material Design 3

## Setup

1. Make sure you have Flutter installed.
2. Clone the repository.
3. Run `flutter pub get` to install dependencies.
4. Run `flutter run` to start the app.

## Project Structure

- `lib/main.dart`: Contains the main application entry point and basic UI structure.
